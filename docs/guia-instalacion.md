# Guía de instalación
- Creo el repositorio en una de mis cuentas y añado como colaborador a mi otra cuenta
- Acepto la colaboracion en mi otra cuenta que es en la que voy a trabajar
- Hago un git clone para clonarme el repositorio en local
- Me creo las diferentes ramas
- En la rama feature/estructura tengo la estructura principal del proyecto
- Inicio las dependencias con npm init -y
- Con npm install jest y npm install jsdoc, los instalo para poder hacer las pruebas y la documentación de manera automática
- Luego me creo la rama release en la que voy a trabajar
- Hago los diferentes cambios en los archivos añadiendo el código correspondientes
